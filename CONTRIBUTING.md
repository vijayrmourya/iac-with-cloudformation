# Contributing to multi-stack-cloudformation-tf-iac-module

Thank you for your interest in contributing! This project provides a hybrid CloudFormation and Terraform IaC reference.

## How to contribute

1. **Report Bugs**: Use the GitHub Issues tracker.
2. **Suggest Features**: Open an issue to discuss new resource templates, security hardening, or CI/CD patterns.
3. **Submit Pull Requests**: 
    - Fork the repository.
    - Create a feature branch.
    - Ensure your changes pass `terraform validate`.
    - If modifying the CloudFormation template, verify it manually or with `aws cloudformation validate-template`.
    - Submit the PR for review.

## Code of Conduct

Please be respectful and professional in all interactions.
